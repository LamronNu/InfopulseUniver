Напишите простую реализацию алгоритма Hash Map с открытой адресацией,
для ключей типа int и значений типа long. Для упрощения размер внутренней
хеш-таблицы может быть фиксированным.

Реализуйте операции put, get, size.

Протестируйте правильность работы с помощью соответствующих Unit Tests.

Вы можете самостоятельно выбрать срок выполнения задания.
